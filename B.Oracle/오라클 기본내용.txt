DBMS : 데이터베이스의 생성 및 구조명시/ 검색,수정,삽입,삭제 
          시스템의 고장이나 권한이 없는 사용자의 접근 등으로부터 데이터를 안전하게 보호하고 관리하는 소프트웨어 패키지


SQL(Structured Query Language)
기본기능 : 입력 삭제 수정 조회
데이터 구조 생성, 접근권한 설정

시험직전 잠깐 볼것 : 
```
DDL 데이터; 정의어
DML 데이터; 조작어 (8~90%)
DCL 데이터; 제어어 아이디 비밀번호 세팅 => xe는 없음
```
클라이언트 ---요청--->서버  : 서버가 사용하는 통신 규격을 따라서 요청을 보내야함.
서버 --- 응답 ---> 클라이언트

oracle - 시스템당 db1개 : 1명의 유저로 로그인하면 해당 db로 접속
mysql - 시스템에서 여러개의 db 생성가능 : 로그인 후 원하는 db로 접속하기 위한 명령어 입력

형 : 레코드(하나의데이터)
열 : 컬럼 혹은 필드
물리적 DB - 실기

논리적 DB - 이론  // 이론과 실기에서 쓰이는 언어가 다름(같은걸 표시하나 다르게 표현함)

동떨어진데이터가 잘못관측된데이터인지 커버리지안의 데이터인지 판단하려면 배경지식이 필요하다. // 도메인지식 : 앞쪽에서 나옴(1과목);경험적지식

테이블의 구조 : 각 데이터의 이름이 명시되고 데이터 타입을 명시
int(number) 정수
varchar(varchar2) 가변길이문자열(최대길이)
char 고정길이문자열
datetime 날짜형식


파이썬에서 데이터 타입을 제한하는방법
def 함수이름(변수값1:int,  변수값2:str) ->str :
	.... return 결과값

변수값1은 int형만 2는 str만 결과값은 str만으로 나타내야함(무시해도 오류발생은 안하지만 경고)
